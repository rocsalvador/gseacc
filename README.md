# gseacc

## Installation

### Requirements
- Rcpp (R package)
- g++

```bash
git clone https://github.com/rocsalvador/gseacc/
make install -j
```

### Documentation

#### C++

For the C++ documentation check this [page](https://rocsalvador.github.io/).

#### R

For the R documentation use ? in R. Available commands:

- ```?Gsea```
- ```?run```
- ```?runChunked```
- ```?filterResults```
- ```?normalizeExprMatrix```
- ```?readCsv```
- ```?readGeneSets```
- ```?writeGeneSets```
